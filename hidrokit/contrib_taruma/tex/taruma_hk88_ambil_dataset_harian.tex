\begin{itemize}
\tightlist
\item
  Serupa dengan isu \#79, akan tetapi dataset merupakan data harian.
\item
  Mengambil dataset harian dalam excel yang berupa \emph{pivot table}.
\item
  Mengubah tabel tersebut ke dalam bentuk \texttt{pandas.DataFrame},
  dengan baris menunjukkan observasi/kejadian dan kolom menunjukkan
  stasiun.
\end{itemize}
